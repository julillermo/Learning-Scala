## Naming Conventions

- Class names: `Person`, `StoreEmployee`
- Variable names: `name`, `firstName`
- Method names: `converToInt`, `toUpper`f

## Scala Features (the ones I selected to highlight)

- Pure Object-Oriented Programming (OOP) langauge. Every variable is an object, and every "operator" is a method
- ALSO a Functional Programming (FP) langauge. Functions are also variables that can get passed into other functions
- You have the option to combine OOP and FP styles in a hybrid way.
- Scala source code compiles to `.class` files that run on the JVM.
- Scala integrates with a lot of Java libraries
- "... you can be productive with it on Day 1, but it's also a deep language so as you go along you'll keep learing,
  and finding newer, better, ways to write code."
- concise, readable code.
