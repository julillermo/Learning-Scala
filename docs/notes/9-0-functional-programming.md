# Functional Programming

- Scala supports both Object-Oriented Programming (OOP) and Functional Programming (FP), even both at the same time in a hybrid style.
- Quick rundown on what to know about Functional Programming
  - Use only **pure functions** (no side-effects) and **immutable values**
  - In a way, it's a really close to how we represent math equations, but in the programming space
- This section of the book will only cover essential parts of functional programming.
